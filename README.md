# Clicker Game version 1.0 :point_up_2:
 유니티 + 뒤끝 서버를 이용한 펫 키우기<br>

# :iphone:프로젝트 소개
간단한 클릭으로 귀여운 펫을 키우기 위한 게임<br>

# :computer:개발 환경
- `Unity 2022.3.10f1`
- External Script Editor : Visual Studio 2019
- 뒤끝 SDK : Backend-5.11.4

# :notebook:주요 기능
- 회원가입
- 로그인
- 게임 데이터 가져오기
- 게임 데이터 수정 및 업데이트
